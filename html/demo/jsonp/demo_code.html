demo下载

demo.zip

必需文件

jquery-1.9.1.min.js 后面的代码直接复制即可运行 

index.html




<body>
	<script type="text/javascript" src="jquery-1.9.1.min.js"></script> 
	  <script>
		function demo(res){
			console.log(res)
		}
	 </script>
	 <script src="./data.json?callback=demo"></script>
</body>





data.json


             --在json文件中 定义demo时 必须要有 （括号） 


demo({
    "pro":[
        {"name":"tom"},
        {"sex":"man"},
        {"job":"web"}
    ]
})




注意：
1.可以使用jsonp进行跨域 不过jsonp只支持get请求
2.引入的文件名需要和json文件中的函数名需要相同 并且引入的json文件需要写在定义函数之上 否则会读取json文件失败
