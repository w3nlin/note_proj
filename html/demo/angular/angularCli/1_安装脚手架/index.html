1.安装 : 
        --npm  i   -g   @angular/cli
        --或 cnpm  i   -g   @angular/cli


2.创建空项目 : 
	ng new 项目名   
	        --例 : ng  new  myngapp01
		配置1:Would you like to add Angular routing?(y/N)  选择y //是否添加路由功能  //大写N代表默认值为N
		配置2:Which stylesheet format would like to use?  选择css //选scss和sass也行，因为sass兼容css

3.启动项目 : 
       （1）myngapp01目录下执行：
       	--npm start或ng serve  
       （2）然后，浏览器地址栏输入：
	--localhost:4200或127.0.0.1:4200
