二：vue-cli
1.安装vue-cli2.0
2.安装element-ui
npm i element-ui -S
3.引入 Element（完整引入）
在 main.js 中写入以下内容：
import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
4.项目根目录新建.babelrc ,内容为:
{
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

7.过程也安装操作了自定义主题，具体操作步骤如下     #去www.w3nlin.xyz的index中查看过程以及注意事项(关键词搜索'element-ui'或'报错'或'vue')
(1)安装工具
npm i element-theme -g
(2)安装白垩主题
npm i element-theme-chalk -D
(3)初始化变量文件
et -i element-variables.scss      #执行成功会在当前目录生成element-variables.scss文件
(4)可在element-variables.scss文件中编辑，如:
$--color-primary: red;
(5)编译-编译主题
et
(6)引入主题
import '当前安装目录的/theme/index.css'     #我是在src目录下安装的，所以import './theme/index.css' ,官网是显示'../theme/index.css'，应该它是安装在项目根目录
import ElementUI from 'element-ui'
import Vue from 'vue'
Vue.use(ElementUI)
